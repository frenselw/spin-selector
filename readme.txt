============================================================
 SpinSelector 幸運轉盤 - 簡易使用說明
============================================================

各位老師好！

這是一個簡單、免費的課堂抽籤小工具，希望能幫助您在課堂上隨機點名、分組，增加互動的趣味性。

------------------------------------------------------------
【如何開始使用？】
------------------------------------------------------------
本工具只有一個 `namepick.html` 檔案，完全不需要安裝！

1.  將 `namepick.html` 檔案存放在您喜歡的任何地方（例如桌面或USB隨身碟）。
2.  直接用滑鼠左鍵【雙擊】這個檔案，它就會在您的瀏覽器（例如 Chrome、Edge 或 Firefox）中打開，可以立即使用。

------------------------------------------------------------
【主要功能介紹】
------------------------------------------------------------
1.  【輸入名單】：
    在右側的「Entries」(輸入項目) 文字框中，輸入學生的名字，每個名字佔一行。

2.  【開始抽籤】：
    直接用滑鼠點擊左側的轉盤，它就會開始旋轉，並自動選出幸運兒。

3.  【管理不同班級的名單】：
    如果您教很多班級，這個功能非常實用！
    - **儲存名單**：在「Current List Name」(目前名單名稱) 中為您目前的名單命名（例如：一年一班），然後點擊「Save」(儲存)。
    - **讀取名單**：從下拉選單中選擇您之前儲存過的班級名單，即可快速載入。
    - **刪除名單**：從下拉選單中選擇一個名單，然後點擊紅色的「Delete」(刪除) 按鈕。

4.  【其他功能】：
    - **Shuffle Names (打亂順序)**：隨機打亂目前名單在轉盤上的位置。
    - **Select Theme (更換主題)**：可以更換轉盤的顏色主題，讓視覺更活潑。

------------------------------------------------------------
【如何在不同電腦間使用？ (例如：在家裡和學校電腦之間同步)】
------------------------------------------------------------
這個工具可以讓您把儲存好的名單帶著走！

步驟一：在您的舊電腦上【匯出】名單
1.  點擊綠色的「Export Lists to File」(匯出名單到檔案) 按鈕。
2.  瀏覽器會提示您儲存一個名為 `spinselector_lists.json` 的檔案。
3.  請將這個檔案和 `namepick.html` 一起存放到您的USB隨身碟中。

步驟二：在您的新電腦上【匯入】名單
1.  在新電腦上，打開USB隨身碟中的 `namepick.html`。
2.  點擊藍色的「Import Lists from File」(從檔案匯入名單) 按鈕。
3.  選擇您剛剛存在USB隨身碟裡的 `spinselector_lists.json` 檔案。
4.  完成！您所有的班級名單就都同步到新電腦上了。
